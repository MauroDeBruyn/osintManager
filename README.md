## ğŸ§­ About <a name="about"></a>

_OSINT Collector_ is a Qt-based desktop application designed for cybersecurity professionals and pentesters to efficiently collect and organize OSINT (Open-Source Intelligence) data. Users can input and manage data from multiple sources, then export it in `.json` or Excel formats for analysis and reporting.

> This tool was created to simplify data management during penetration tests, threat assessments, and digital investigations, providing a structured approach to OSINT collection.
